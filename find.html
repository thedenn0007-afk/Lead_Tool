Step 2 — Scrape with Free Extension
Install once, use forever. No coding.
⚡
Instant Data Scraper
Free Chrome extension · Auto-detects tables · 1-click export
Install Free →
1
Open Google Maps using the query you built above. Scroll the left results panel until 80–200+ businesses are loaded.
2
Click the Instant Data Scraper icon in your Chrome toolbar.
3
The extension auto-detects the business list. Click Start Crawling — it scrolls and collects automatically.
4
When done, click Download CSV. You'll get a file with: Name, Address, Phone, Rating, Website, Maps Link.
5
Come back here and import that CSV file below. All leads go into your database instantly.
Tip: One search = 80–150 businesses. Do 5–10 different area searches = 500–1000 leads in one hour.
Alternative — Python Local Scraper
Use the same search query, download the script, then run one command in Terminal / CMD.
1
Build your search on the left. The command below updates automatically from those fields.
2
Download gmaps_scraper.py and save it in a folder on your computer.
3
Install once in Terminal / CMD: pip install playwright pandas then playwright install chromium
4
Copy the generated command below and run it in Terminal / CMD from the same folder.
Run this command in Terminal / CMD
python gmaps_scraper.py "dental clinic bangalore" 200
Download Python Script
Important: A normal browser page cannot run Python on your computer directly. It can prepare the command and download the script, but the actual run must happen in Terminal / CMD unless you build a desktop app or backend service.
