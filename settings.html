ℹ️ About LeadTool

LeadTool is a free, zero-dependency lead generation and follow-up tool for anyone selling websites to local businesses.

No API keys. No backend. No login. All data stays in your browser.

Pages: Find Leads · Auto Scrape · Guide · Settings

🔧 How Data is Stored

Everything is saved in your browser's localStorage.

✅ Works offline

✅ No server needed

⚠️ Data is browser-specific — doesn't sync between devices

⚠️ Clearing browser data will erase leads — export CSV regularly!
